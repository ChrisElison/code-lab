# 🇨 C Examples & Projects

Here's some of my C example projects, tests and exercises. I am not very experienced in C, I am still
not quite sure of the point of pointers, but I will keep going and learning. I want to get more into
writing Raylib programs in C, so expect more of those in particular!

As always, this code may have bugs, errors or use inefficient techniques, I'm an amateur - sorry!

## Contents:
* 📁 **include**
    * This directory holds important library header files that are included in other projects
* 📁 **ircd**
    * My first attempt at vibecoding an IRC server, it currently only accepts connections, nothing else
* 📁 **lib**
    * This directory holds library files to be included in other projects
* 📁 **randnum**
    * Simple number guessing game. Generates a random number and then you try to guess it, if you run out of attempts it's game over!
* 📁 **raylibtest**
    * Testing if Raylib works - it does!
* 📁 **rev-string**
    * Simple program that accepts a string of characters then prints it reversed
