# 🐍 Python Examples & Projects

I started learning Python so I could write some examples and exercises for my niece who is doing Python
programming for her GCSEs, in fact this whole Code Lab repository idea grew from these Python examples.

I quite like the language, it's very easy to do stuff that would ordinarily take a lot of lines to do in 
other languages like C or Java.
