# ⚡ Arduino Code

Various Arduino sketches I've written (or attempted to) since starting my electronics hobby
all the way back in 2016/2017. A lot of these sketches may be incomplete or have errors, you
have been warned! I have tried to comment some of the code, but some sketches have no comments.

I'm not sure if this is everything, I have had three different PCs since 2016, I may well have
lost some sketches to the void, sorry about that.

If you are looking for code from ChrisElisonElectronics.Com, that will be in it's own 
repository, although I'll have it here too.

Most of this code is written for either Arduino Uno/Nano or the ESP8266.
